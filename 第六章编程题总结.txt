这道题考虑的点真不是一般的多，总结如下：
1:当指数为1时，不显示1
如：
1 1
0 0
0 0
输出 x
2：当系数为1时不输出1，但要保留+
如：
3 1
2 1
1 1
0 1
0 0
输出 x3+x2+x1+1
3：当系数为-1时不输出1，但要保留-
3 -1
2 -1
1 -1
0 -1
1 -1
0 0
输出 -x3-x2-2x-1
4：当全为0时，只输出 0
如
0 0
0 0
输出 0
5：当系数为负数时不输出+
例 同3
6：最高次幂项不能输出+
7：系数为0的非0次幂不输出
如
2 1
1 -1
0 1
1 1
0 1
输出 x2+2

其实就是完全按照我们日常的书写习惯，写程序时有些情况可能想不到就过不了。
